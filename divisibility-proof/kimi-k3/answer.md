命题



对任意正整数 $n$，$6 \mid n^3 - n$。




关键观察（两条路径共同的基础）：

$$n^3 - n = n(n^2-1) = (n-1)\cdot n \cdot (n+1)$$

即 三个连续整数之积。





证明一：数学归纳法



基础步骤：当 $n=1$ 时，$n^3 - n = 1 - 1 = 0 = 6 \times 0$，命题成立。




归纳假设：设 $n = k$（$k \geq 1$）时命题成立，即 $6 \mid k^3 - k$。




归纳递推：考察 $n = k+1$：

$$(k+1)^3 - (k+1) = k^3 + 3k^2 + 3k + 1 - k - 1 = (k^3 - k) + 3k(k+1)$$




分别分析两项：





由归纳假设，$6 \mid k^3 - k$；


对第二项：$k$ 与 $k+1$ 是相邻整数，必为一奇一偶，故 $k(k+1)$ 是偶数。记 $k(k+1) = 2m$，则 $3k(k+1) = 6m$，即 $6 \mid 3k(k+1)$。





两项均被 6 整除，故 $6 \mid (k+1)^3 - (k+1)$。




由数学归纳法，命题对所有正整数 $n$ 成立。$\blacksquare$





证明二：组合论（算两次）



设 $S$ 是含 $n+1$ 个元素的集合。计数对象：从 $S$ 中取出 3 个互不相同的元素排成一列的有序三元组 $(a,b,c)$ 的总数 $N$。




第一种数法（直接排列）：

$a$ 有 $n+1$ 种取法，$b$ 有 $n$ 种取法，$c$ 有 $n-1$ 种取法，故

$$N = (n+1)\cdot n \cdot (n-1) = n^3 - n$$




第二种数法（先选后排）：

先从 $S$ 中选定一个 3 元子集，选法数为组合数 $\binom{n+1}{3}$（它计数子集的个数，必为非负整数）；再将选出的 3 个元素排入三个位置，有 $3! = 6$ 种排法。故

$$N = 6 \cdot \binom{n+1}{3}$$




比较两式：

$$n^3 - n = 6\cdot\binom{n+1}{3}$$




右端显然是 6 的倍数，故 $6 \mid n^3 - n$。

（$n=1$ 时 $S$ 仅 2 个元素，不存在 3 元子集，$\binom{2}{3}=0$，对应 $1^3-1=0$，结论仍成立。）$\blacksquare$






注：此证明的实质是"排列数 = 组合数 × 阶乘"——因子 $3! = 6$ 从"先选后排"的对称性中自动涌现。它同时揭示了为何"任意三个连续整数之积必被 6 整除"。







证明三（附加）：因式分解 + 逐素因子分析



由 $n^3 - n = (n-1)n(n+1)$，只需分别证它被 2、3 整除（因 $\gcd(2,3)=1$）：








因子 2：$n-1,\ n,\ n+1$ 中含相邻整数，相邻整数必有一偶，乘积含因子 2；








因子 3：按 $n \bmod 3$ 分类讨论：





$n \equiv 0 \pmod 3 \Rightarrow 3 \mid n$；


$n \equiv 1 \pmod 3 \Rightarrow 3 \mid n-1$；


$n \equiv 2 \pmod 3 \Rightarrow n+1 \equiv 0 \pmod 3 \Rightarrow 3 \mid n+1$。





任何情形下乘积都含因子 3。








故 $2 \times 3 = 6$ 整除 $n^3 - n$。$\blacksquare$
