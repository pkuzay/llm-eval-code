# GPT-5.6 Answer

**命题：** 对任意正整数 (n)，都有
[
6\mid (n^3-n).
]

## 证明一：数学归纳法

**第一步：验证基础情形。**

当 (n=1) 时，
[
1^3-1=0,
]
显然 (0) 能被 (6) 整除，因此命题对 (n=1) 成立。

**第二步：归纳假设。**

假设当 (n=k) 时命题成立，即存在整数 (m)，使得
[
k^3-k=6m.
]

**第三步：证明 (n=k+1) 时命题也成立。**

计算：
[
\begin{aligned}
(k+1)^3-(k+1)
&=k^3+3k^2+3k+1-k-1\
&=(k^3-k)+3k^2+3k\
&=(k^3-k)+3k(k+1).
\end{aligned}
]

根据归纳假设，(k^3-k) 能被 (6) 整除。

又因为 (k) 与 (k+1) 是两个连续整数，其中必有一个是偶数，所以
[
k(k+1)
]
一定能被 (2) 整除。因此
[
3k(k+1)
]
一定能被 (6) 整除。

所以
[
(k+1)^3-(k+1)
]
也能被 (6) 整除。

由数学归纳法可知，对任意正整数 (n)，
[
n^3-n
]
均能被 (6) 整除。

---

## 证明二：组合论法

考虑从 (n+1) 个不同元素中，依次选出三个互不相同的元素，并且考虑它们的排列顺序。

第一位置有 (n+1) 种选择，第二位置有 (n) 种选择，第三位置有 (n-1) 种选择，因此这样的有序排列共有
[
(n+1)n(n-1)
]
个。

展开可得
[
(n+1)n(n-1)=n(n^2-1)=n^3-n.
]

另一方面，我们也可以先从 (n+1) 个元素中选出三个元素，共有
[
\binom{n+1}{3}
]
种选择。每组选出的三个不同元素都可以按照
[
3!=6
]
种顺序排列。

因此有序排列的总数为
[
6\binom{n+1}{3}.
]

于是
[
n^3-n=6\binom{n+1}{3}.
]

由于 (\binom{n+1}{3}) 是整数，所以 (n^3-n) 一定能被 (6) 整除。

综上，对任意正整数 (n)，都有
[
\boxed{6\mid(n^3-n)}.
]
