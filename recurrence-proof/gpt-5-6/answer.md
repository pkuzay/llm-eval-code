# GPT-5.6 Answer

递推关系为

[
a_n=3a_{n-1}-2a_{n-2},\qquad a_1=1,\ a_2=3.
]

## 一、求通项公式

对应的特征方程是

[
r^2-3r+2=0.
]

因式分解：

[
(r-1)(r-2)=0,
]

所以两个特征根为 (r=1,2)。因此通解为

[
a_n=A\cdot 1^n+B\cdot 2^n=A+B2^n.
]

代入初始条件：

[
\begin{cases}
A+2B=1,\
A+4B=3.
\end{cases}
]

两式相减得

[
2B=2\Rightarrow B=1,
]

进而

[
A+2=1\Rightarrow A=-1.
]

所以通项公式为

[
\boxed{a_n=2^n-1}.
]

## 二、用数学归纳法证明

证明对所有 (n\ge 1)，都有

[
a_n=2^n-1.
]

**基础步骤：**

当 (n=1) 时，

[
a_1=1=2^1-1.
]

当 (n=2) 时，

[
a_2=3=2^2-1.
]

结论成立。

**归纳步骤：**

假设对于某个 (k\ge2)，有

[
a_k=2^k-1,\qquad a_{k-1}=2^{k-1}-1.
]

由递推关系，

[
\begin{aligned}
a_{k+1}
&=3a_k-2a_{k-1}\
&=3(2^k-1)-2(2^{k-1}-1)\
&=3\cdot2^k-3-2^k+2\
&=2\cdot2^k-1\
&=2^{k+1}-1.
\end{aligned}
]

因此，当结论对 (k-1) 和 (k) 成立时，对 (k+1) 也成立。

由数学归纳法可知，对所有 (n\ge1)，均有

[
\boxed{a_n=2^n-1}.
]
